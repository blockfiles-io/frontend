About

This project has been created privately by me, Ralph Kuepper (@amlug), in my own private time. If this project gains traction and people wnat more features this can grow into a lot more. Also if you are interested in helping in this project please reach out at info@blockfiles.io. The following roadmap is just meant to provide some gudelines, I am not committing to doing any of it in any specific timeframe:

Why blockfiles.io?

I noticed that I often don't even know when something happens in my (many) wallets. Yes, some wallets (TrustWallet) send out push notifications but I don't have all my wallets in there. And I don't want to have them all in there. So I simply want to know what happens with my wallets in an easy way. And that's what blockfiles.io does. And nothing more (for now).

Roadmap / Milestones

The following steps are all based on the number of addresses being monitored. The more addresses are being monitored the more sense it makes to continue this project.

Less than 100 addresses (NOW)
  • Everything for free, no ads.
  • Email only
  • Polygon Mainnet
  • Ethereum Mainnet

100 addresses
  • Allow text notifications (within limits)
  • Create twitter account
  • Dedicate more time to it
  • Build proper design for it

1000 addresses

  • Start thinking about monitizing in a web3 way
